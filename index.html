              1.4.2 2026-07-15
              • Tighter sidebar spacing — reduced gaps between smart views, Focus section (In Progress / Paused), and their items for a more compact left panel
              • Sync now and Settings buttons are now side by side on the last row when Dropbox is connected
              • Right detail panel now has a "Mark complete" button (with "Mark incomplete" for already-done tasks), making it usable from the panel without returning to the list
              1.4.1 2026-07-15
              • Fixed crash on corrupted task data — JSON.parse on state load now wrapped in try/catch so a broken localStorage entry no longer kills the entire app
              • Fixed tag filter dropdown label not updating when clicking a tag chip on a task
              • New default list name is now "Today" instead of "Default" for first-time visitors
              1.4.0 2026-07-15
              • New "Paused" status — set a task aside without losing its place. A dedicated "Paused" sidebar section (right below In Progress) lists and counts paused tasks
              • Toggle Paused from the right-click menu or the task detail panel; click the "⏸ Paused" chip to send a task back to To Do. Completing a paused task clears the status, like In Progress
              1.3.2 2026-07-14
              • Fixed Dropbox "Connect to Dropbox" button not responding — script was aborting early due to AGGREGATE_VIEWS.has (arrays don't have .has()); also hardened dbx state parsing against corrupted localStorage
              1.3.1 2026-07-10
              • Right-click menu "Due Today" option — sets the task's due date to the current day; hidden if already due today
              • Completing a task now clears "In Progress" status — no longer shows the chip or counts toward the sidebar
              1.3.0 2026-07-10
              • Right-click context menu on tasks — "Set In Progress" and "Skip" (for recurring tasks) actions directly on any task
              • "Skip" on a recurring task advances the due date to its next occurrence and clears the "In Progress" status if set
              1.2.0 2026-07-09
              • Smooth animated transitions when adding, completing, or deleting tasks — tasks fade and slide in/out instead of popping
              • Contextual empty states with themed SVG illustrations — different artwork and messages for empty lists, smart views, search results, and filtered views
              • Sticky group headers (Overdue, Today, Tomorrow…) stay pinned while scrolling through that group; a subtle shadow appears at the top of the list when scrolled
              • Deeper glass hierarchy — modals and the Quick Open palette now have stronger blur for a more layered HUD feel; theme colour changes transition smoothly across the shell, sidebar, and detail panel
              • Natural Language Quick Add — type #tags, !priority, and due dates (tomorrow, next week, etc.) right in the task input
              • Quick Open palette — press Cmd+K / Ctrl+K to search every task, description, and tag, then jump straight to it
              1.1.4 2026-07-08
              • Default list renamed from "Default" to "Today" — new visitors now start with a "Today" list instead of a generic "Default" list
              • "In Progress" sidebar section with its own glass container, count badge, and icon — sits between smart views and your lists
              1.1.3 2026-07-07
              • Settings page redesigned with larger cards, bigger swatches, softer glass, and more spacing
              • Primary buttons now use your theme accent colour instead of a generic tint
              • Toggle switches, mode picker, tabs, and badges all polished for a cleaner look
              1.1.2 2026-07-07
              • Light/dark mode can now follow your system preference — pick "System" in settings
              • Date chips in the task list now use your theme colour instead of a generic tint
              • Reset filters button placed conveniently on the right of the "Filters & sort" toggle
              1.1.1 2026-07-06
              • Filters & sort are now tucked into a collapsible section with a badge showing how many filters are active
              • Fixed sidebar and task-detail panel resize handles not responding to touch on tablets
              • Fixed dialogs and toast notifications staying neutral grey instead of following your theme colour
              1.1.0 2026-07-05
              • Custom glass-themed dropdowns: filter, sort, list, and recurrence selects now use tinted glass panels matching your theme colour
              • Due date now uses a custom glass-themed calendar picker instead of the browser's default one
              • Fixed dropdown menus staying too dark and losing their theme colour in light mode
              • Fixed open dropdowns letting clicks pass through to controls behind them
              • Drag-to-reorder tasks in the main list (when sorted manually)
