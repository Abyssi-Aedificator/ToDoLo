          Changelog

          0.1.0
          • Initial release: glassmorphism to-do app with multi-list support
          • Task due dates, recurrence, priority, description, and sub-tasks
          • "All Tasks" aggregated view and auto-save task details panel
          • Recurrence can only be set when a due date is present
          • Description now saves on blur instead of every keystroke
          • Sub-task chip shows completion percentage (e.g. "5/10 - 50%")
          • Settings page with version and changelog
          • Active task filter (All/Active/Done) is remembered between sessions
          • Due date and priority filters, combinable with the All/Active/Done filter
          • Completed tasks always sort to the bottom of the list
          • New tasks are added to the top of the list and open their detail panel automatically
          • Completing the task shown in the detail panel closes the panel
          • Up to three mixable sort criteria with independent direction and a reset button
          • Date-based group separators (e.g. "June 2026") when sorting by due date, priority, etc.
          • Smart sidebar views: All, Today, Tomorrow, and Next 7 Days
          • Smart views moved into their own card with distinct colored icons and a "Lists" heading above your lists
